2️⃣ Run tests by tags

Run only smoke tests:

radish -t smoke features/


Run regression tests:

radish -t regression features/


Run tests with multiple tags:

# AND condition (must have both tags)
radish -t regression,negative features/

# OR condition (any tag)
radish -t "smoke,regression" features/

3️⃣ Run tests in parallel

Use the -p or --parallel option:

# Run all features in 3 parallel processes
radish -p 3 features/


Combine tags + parallel execution:

# Run only regression tests in 3 parallel processes
radish -t regression -p 3 features/
